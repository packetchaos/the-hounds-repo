🧠 Advanced search — cross-table NL→SQL with automatic joins

Ask anything across the whole of navi.db and the model writes the joins for you. e.g. "assets in the Production tag with the highest EPSS score" · "critical vulns with EPSS over 0.5 on cloud assets" · "hosts running OpenSSL that also have an expiring certificate". It drafts the SQL with joins, you review/refine, then execute (read-only).

Advanced search needs an LLM. Set ANTHROPIC_API_KEY in the server environment and reload. You can still hand-write SQL on the explorer pages.
