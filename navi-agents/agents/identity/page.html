🆔 Identity inventory

Discovers identities by content, not a memorized plugin list — sweeps plugin names and raw output, then the high-value enumerators (passwd 95928, SMB users 10860, non-expiring 83303, host SID 10859, SNMP 41028). Classifies Human / Service-NHI / Machine — built-ins are labelled by external authority (Microsoft well-known SIDs/RIDs, AD machine $ accounts, the Linux login.defs UID convention), not just our guess lists — flags the risky ones (non-expiring, privileged, guest, default-secret), and headlines the coverage gap — hosts scanned without credentials. Tagged via navi tag --query; gated writes.
