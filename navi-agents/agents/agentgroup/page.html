🛰️ Agent Group Tagging

Sirius tags every asset in a Tenable agent group with Agent Group : <name> — runs navi enrich tag --c "Agent Group" --v "<name>" --group "<name>" per group, on the background queue (see the Tagging log). Three ways below. Writes are gated.

① Tag by a list of group names

Paste group names (one per line or comma-separated). Each is matched to a real agent group and tagged.

② Pick groups to tag

Tick the agent groups, then tag the selected.
